# CatchBan 2.0 Beta

> 捕获灵感，而后掌控。  
> Catch the spark. Command the workflow.

CatchBan 2.0 Beta 是面向视频创作者的 macOS 素材库工作台。  
它把采集、管理、预览与轻创作整合到同一条本地工作流中，让素材从“下载完成”直接进入“可创作状态”。

## 现在发布

- 版本: `V2.0 Beta`
- 状态: 公开测试中，完全免费
- 平台: macOS 12+（Apple Silicon 原生推荐，同时兼容 Intel）
- 更新时间: 2026 年 4 月

## 核心能力全景

### 精准采集

- 链接粘贴与内置浏览器双入口采集
- 支持 4K 高清视频下载（视源站可用性）
- 下载结果可直接进入素材库，减少二次搬运

### 灵活归档

- 本地 Creator Library Workspace（新建或接入已有目录）
- 支持按项目/主题/阶段组织素材
- 覆盖全库搜索、最近记录、稍后整理（Inbox）等常用入口
- 提供重复候选、同来源归档与元数据补全建议，降低素材混乱

### 原生级检视

- 在库内快速预览，减少 Finder 与外部播放器来回切换
- 支持逐帧查看、重点片段回看与标记动作
- 本地优先链路，兼顾高频检视时的稳定与响应

### 创作工具箱

- 无损转换为 H.264（Compatible MP4）
- 截取片段
- 抓取关键帧
- GIF 导出
- 音频提取
- 批量缩略图抓图

## V2.0 Beta 版本升级重点

- 系统级产品升级：CatchBan 从单点下载工具升级为「创作者素材库工作台」，把采集、整理、预览与输出整合进同一条工作流
- 采集能力升级：支持 4K 高清视频下载与多种清晰度选择，并通过内置浏览器辅助验证处理更复杂的采集场景
- 素材库体系升级：下载内容、本地视频与导出结果统一归档到本地工作台，支持按项目、主题、阶段持续整理与复用
- 检视体验升级：预览、逐帧查看与关键帧标记链路进一步优化，减少在 Finder、播放器与编辑器之间来回切换
- 创作工具升级：新增或强化视频转码、片段导出、GIF 导出、关键帧抓取、音频提取与批量缩略图等高频前处理能力
- 工作流智能升级：加入全库搜索、最近记录、稍后整理（Inbox）、重复候选与同来源归档建议，更适合大体量素材管理
- 后期衔接升级：选定素材可直接进入 Premiere Pro、DaVinci Resolve、Final Cut Pro 等后期流程，缩短从参考到输出的距离
- 升级连续性保障：现有本地素材库结构与资产可延续到后续版本，Beta 阶段继续免费开放并持续迭代
