# CatchBan: The Creator's Workspace

CatchBan is a native macOS workspace for collecting, organizing, previewing, and preparing video references & source assets.  
CatchBan 是一款面向创作者的 macOS 原生素材工作台，用于采集、整理、预览与预处理视频参考素材和源素材。

Built for creators who need a fast local workflow instead of scattered tools and folders.  
它面向需要高效本地工作流的创作者，帮助你摆脱分散的软件与混乱的文件夹。

---

## ✨ Core Capabilities

### 🗂 Creator Library Workspace
- Organize downloaded clips, local videos, and exported results in one unified workspace.
- 将下载内容、本地视频与导出结果统一纳入同一个素材工作台。
- Structure assets by project, topic, or production stage.
- 支持按项目、主题或工作阶段组织素材。

### 🎥 Video Capture
- Capture references through direct links and browser-assisted workflows.
- 支持通过链接与浏览器辅助方式采集视频参考素材。
- Supports high-quality downloads where the source allows it.
- 在源站支持的前提下可获取高质量素材。

### 👀 Preview & Review
- Preview assets directly inside the workspace.
- 可直接在工作台内预览素材。
- Review clips frame by frame and inspect key moments faster.
- 支持逐帧检视，更高效地筛选重点镜头与片段。

### 🧰 Creative Utility Tools
- H.264 conversion
- H.264 转码
- Clip export
- 片段导出
- GIF export
- GIF 导出
- Frame capture
- 关键帧抓取
- Audio extraction
- 音频提取
- Batch thumbnail generation
- 批量缩略图生成

### 🧠 Library Intelligence
- Global search, history, Inbox, duplicate candidates, and same-source grouping suggestions.
- 支持全库搜索、历史记录、Inbox、重复候选与同来源归档建议。

### 🎬 Post-Production Ready
- Move selected assets into editing workflows faster.
- 更快将选定素材衔接到后期工作流。
- Designed to reduce the distance between collecting references and starting the edit.
- 旨在缩短从“收集参考”到“进入剪辑”的路径。

## 💾 Local-First by Design

- Your media and library data stay under your control.
- 你的素材与素材库数据默认由你自己掌控。
- CatchBan is built around a local workflow rather than a cloud-dependent one.
- CatchBan 以本地优先工作流为基础，而不是依赖云端同步。
- Libraries can live on your internal drive, external SSD, or mounted storage.
- 素材库可建立在系统盘、外接 SSD 或已挂载存储设备上。
